Dato

Son la representacion simbolica ya sea mediante numeros o letras de una recopilacion de informacion la cual puede ser cualitativa o cuantitativa.
Se usan para proveer informacion pero por si solos no tienen sentido.

Información

Son un conjunto de datos entrelazados mediante conectores para formar una idea o definición.

Base de datos

Es un conjunto de datos entrelazados mediante conectores lingüísticos para formar una idea o definición.

Servidor de Base de Datos

Son los que se utilizan para los gestores de bases de datos y que varios usuarios hagan varias operaciones sobre ellas situados en lugares diferentes.
Se pueden acceder a ellas mediante terminales o equipos con un programa llamado cliente.

Gestor de Base de Datos

Son un conjunto de programas que manejan todo acceso a la base de datos con el objetivo de servir de interfaz entre esta, el usuario y las aplicaciones utilizadas.

Esquema de Base de Datos

Representa la configuración logica de todo o parte de una base de dato relacional. Puede existir de dos formas: como representación visual y como un conjunto de formulas conocidas como restricc- iones, de integridad que controlan una base de datos.
Estas formulas se espresan en un lenguaje de definición de datos, tal como SQL (Structured Query Language).
